# Project Alpha v0.1

Første frontend-prototype for Project Alpha.

## Innhold
- Moderne mørkt dashboard
- Project Alpha-identitet
- Porteføljeoversikt
- Trading- og langsiktige posisjoner
- AI-score og konfidensscore
- Kandidatliste
- Responsivt design for mobil og desktop

## Kjør lokalt
1. Installer Node.js 20+
2. Kjør `npm install`
3. Kjør `npm run dev`
4. Åpne `http://localhost:3000`

## Foreslått GitHub-oppsett
Opprett et privat repository med navnet `project-alpha`.
Pakk ut prosjektet, åpne terminal i mappen og kjør:

```
git init
git add .
git commit -m "Initial Project Alpha v0.1"
git branch -M main
git remote add origin <DIN_GITHUB_REPOSITORY_URL>
git push -u origin main
```
